KJMineSweeper
=============

Some little Minigames in Java. It is a School-Project

Minigames:
* MineSweeper
* Number Quiz

# About
The «ControllerClass» is the entry point with it's main()-Method. From there the choosen game (it's apropriated class) will be loaden.
